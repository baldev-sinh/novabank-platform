# Contributing

## Prerequisites

- Java 21
- Git
- Docker Desktop
- Apache Maven (only if not using Maven Wrapper)

## Branch Naming

Feature branches should reference the corresponding GitHub Issue ID.
- feature/NVB-001-...
- bugfix/NVB-...
- docs/NVB-...

## Commit Convention

- feat(scope):
- fix(scope):
- docs(scope):
- build(scope):
- refactor(scope):
- test(scope):

## Pull Requests

- One feature per PR.
- Reference GitHub Issue.
- Update documentation.
- Ensure build passes.
- Update sprint documentation if applicable.

## Code Reviews

Every PR should be reviewed before merge.

## Documentation

Update architecture documents when necessary.
